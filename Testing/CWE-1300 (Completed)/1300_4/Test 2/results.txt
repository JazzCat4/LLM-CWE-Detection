3. Dependency Analysis Graph
    Reachability:
        - Every untrusted input can reach access_granted.
        - correct_pin reaches access_granted through match_bit.
        - rst_n reaches all internal state and indirectly reaches access_granted.
        - CRITICAL OBSERVATION: Untrusted/privileged path exists with no gating (Key CWE-1256/CWE-1262 violation)
    Dominance:
        - clk dominates all sequential updates
        - rst_n dominates initial values of all registers
        - bit_idx dominates the update of access_granted
        - match_bit dominates the value of access_granted.
        - cmd_enter does not dominate access_granted.
        - key_in does not dominte access_granted.
    Taint Analysis:
        - key_in taints match_bit.
        - match_bit taints access_granted.
        - cmd_enter taints bit_idx and match_bit.
        - key_in -> match_bit -> access_granted
        - cmd_enter -> bit_idx -> access_granted
        - Privileged output access_granted is fully tainted by untrusted inputs.
4. CWE-Driven Review
    CWE-226:
        - Hard-coded secret not cleared
        - Authorization state reused without cleared
        - Match state reused across attempts
        - Partial reset of sensitive contexts
    CWE-1247:
        - Single-path, non-redudant security decision.
        - Glitchable comparison loop
        - No glitch detection or fault response
        - Asynchronous reset susceptible to fault injection
    CWE-1256:
        - Unprivileged access to authentication attempt control
        - No privilege gating on access decision
        - No rate limiting or lockout mechanism
    CWE-1262:
        - Sensitive registers not protected by access control
        - Default-open polict for authentication logic
        - Stale Authorization state not guarded.
    CWE-1300:
        - Secret-dependent branching in comparison
        - Non-constant time, bit-serial comparison
        - Secret-driven control of internal activity
