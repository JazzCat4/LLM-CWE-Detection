TYPE CLASSIFICATION
Module type: Clock / power /reset logic
Relevant CWEs: 1234, 1247

ASSET IDENTIFICATION
Assets: Shift register contents, output flag (detect)
Privilege-domain boundaries: Input domain -> module domain. Module domain -> output domain. No explicit priv./unpriv. partitioning
Clk and reset: Sequential logic triggered on POSEDGE CLK. Active-low reset. On reset, shift_reg is forced to 0.
Adversary capabilities: Manipulate input, reset, clock
Security rules. None
States / transitions: Implicit FSM via shift_reg. Transition on each posede clk. Detect = 1 if certain state.

All internal state is directly derived from untrusted input.
Output is fully attacker-controlled through taint propagation.
Clock and reset dominate state updates; glitching either can force or prevent state transitions
No illegal states exist, no security rules, priv. boundaries.
