TYPE CLASSIFICATION
Module type: Clock / power / reset logic
Relevant CWEs: 1234, 1247

ASSET IDENTIFICATION
Assets: count (register), state, value (input), LED (output)
Privilege domains: No explicit priv. domains
Clock and reset: Single synchronous process on POSEDGE CLK. Active low reset. On reset, FSM returns to state 0, count=0.
Adversary capabilities: Manipulate inputs, toggling enable input, manipulating CLK/RST, observing LED output
Security rules: No explicit security rules
FSM: State 0 - 3 defined and transition rules. No recovery logic for state > 2 or random corruption.

Reachability: All internal nodes are reachable from external inputs under normal operation
Dominance: The control structure is strictly hierarchical: RST->S0->S1->S2->S0
Taint: All untrusted inputs propagate taint into internal state and outward through LED.
